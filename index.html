📦 1500개로 확장하는 방법
Step 1. 아래 링크에서 N4 단어 데이터 다운로드
👉 github.com/elzup/jlpt-word-list
→ n4.json 파일 다운로드

Step 2. 이 HTML 파일의 const WORDS = [...] 부분을 찾아서
다운받은 단어 데이터로 교체

Step 3. 형식 맞추기
각 단어 항목이 아래 형식이어야 해요:
{k:'단어', r:'읽기', m:'뜻', t:'품사', ej:'예문JP', ek:'예문KR'}

또는 Claude에게 "N4 단어 데이터를 이 형식으로 변환해줘"라고 부탁하면 돼요 😊
📅 하루 25개 모드 동작 방식
• 매일 순서대로 25개씩 배정돼요
• 1일차: 1~25번, 2일차: 26~50번, ...
• 날짜가 바뀌면 자동으로 다음 25개로 넘어가요
• 오답은 오답 복습 모드에 쌓여요
• 300개 기준 12일 완성 → 1500개면 60일(약 2달)
